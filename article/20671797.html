# 前言

Atom 的功能依赖于拓展，但是很多人在安装插件的时候会遇到安装失败，下载失败等问题。其实原因很简单，Atom 的插件源全是国外的链接，部分运营商可能被墙了，因此，需要一定的步骤方可实现正常安装。

# 解决方法

最后通过 apm install --check
如果出现
Checking for native build tools done
意味着可以顺利安装插件了！
